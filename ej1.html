PRÁCTICA 6: Manejo de Bases de Datos con PHP

Ejercicio 1: Completar

Consulta a una base de datos: Para comenzar la comunicación con un servidor de base de datos MySQL, es necesario abrir una conexión a ese servidor. Para inicializar esta conexión, PHP ofrece la función mysqli_connect

Todos sus parámetros son opcionales, pero hay tres de ellos que generalmente son necesarios: $hostname, $nombreusuario y $contraseña

Una vez abierta la conexión, se debe seleccionar una base de datos para su uso, mediante la función mysqli_select_db

Esta función debe pasar como parámetro el nombre de la conexion obtenida previamente y el nombre de la base de datos que se desea utilizar

La función mysqli_query () se utiliza para ejecutar una consulta SQL (como SELECT, INSERT, UPDATE o DELETE) sobre la base de datos especificada

y requiere como parámetros el identificador de la conexión y la cadena de caracteres que contiene la consulta SQL

La cláusula or die() se utiliza para capturar un posible error en la ejecucion, deteniendo el script y mostrando un mensaje de advertencia

y la función mysqli_error () se puede usar para devolver una cadena que describe el ultimo mensaje de error ocurrido en la operacion mas reciente con la base de datos
