Practica 2 - CSS

EJERCITACIÓN

1. ¿Qué es CSS y para que se usa?

CSS son las siglas de Cascading Style Sheets (Hojas de Estilo en Cascada) Es una tecnología utilizada para controlar la apariencia de una página web, permitiendo separar la información (contenido en HTML) del diseño o formato Se usa para especificar aspectos visuales como:

  • Fuentes y tamaños de texto
  • Colores de fondo y de fuente
  • Espaciado entre textos y recuadros (márgenes y rellenos)
  • Posicionamiento de elementos (texto e imágenes) en la página

2. CSS utiliza reglas para las declaraciones de estilo, ¿cómo funciona?

Las reglas de estilo funcionan mediante una sintaxis especial compuesta por tres partes fundamentales

  • Selector: Indica al navegador qué parte del documento se verá afectada (por ejemplo, una etiqueta como < h1 >)
  • Propiedad: Especifica qué aspecto visual se va a cambiar (por ejemplo, color)
  • Valor: Establece la configuración específica para la propiedad (por ejemplo, red)

Funcionamiento: Las propiedades se agrupan dentro de llaves {} y cada una debe terminar con un punto y coma (;) para separarla de la siguiente

3. ¿Cuáles son las tres formas de dar estilos a un documento?

Es posible asociar reglas de estilo a un documento HTML de tres maneras

  • Nivel de elemento (Inline): Se definen directamente sobre la etiqueta HTML usando el atributo style (ej. < h1 style="...">) Es la forma menos recomendada
  • Nivel de página (Interno): Se escriben dentro de la sección < head > del documento, encerradas entre las etiquetas < style > y < /style >
  • Archivo externo: Se agrupan las reglas en un archivo independiente con extensión .css Este archivo se vincula al HTML mediante la etiqueta < link rel="StyleSheet" ... > en el encabezado

4. ¿Cuáles son los distintos tipos de selectores más utiliados? Ejemplifique cada uno

Los selectores mas comunes son:

  • Selector de etiqueta (Tipo): Afecta a todos los elementos de una misma marca HTML. Ejemplo p { color : red; }
  • Agrupamiento de Selectores: Permite aplicar la misma regla a varias etiquetas a la vez separándolas por comas. Ejemplo: h1,h2,h3 { color: red; }
  • Selector de ID: Se utiliza para aplicar estilos a un único elemento específico que posea un identificador único. En el código se representa con un numeral (#). Ejemplo: #recuadro { background-color: yellow; }
  • Selector de Clase: Permite dar a los elementos un nombre identificativo que puede ser usado por varios elementos para compartir un estilo . Se representa con un punto (.). Ejemplo: .editor-note { font-size: 1.2em; }

5. ¿Qué es una pseudo-clase?¿Cuáles son las mas utilizadas aplicadas a vínculos?

Las pseudo-clase se utilizan para definir el estilo de un elemento en un estado especial (como cuando el muouse pasa por encima). Las mas utilizadas en vinculos suelen ser: :link, :visited, :hover, :active

6. ¿Que es la herencia?

Herencia hace referencia a que los "hijos" dentro de la estructura HTML adoptan automaticamente ciertas propiedades de estilo definidas en sus elementos (como el tipo de letra o el color definido en el body).

¿En qué consiste el priceso denominado cascada?

El proceso de cascada es el mecanismo que describe cómo se aplican y priorizan los estilos cuando existen múltiples definiciones para un mismo elemento . Consiste en la capacidad de definir estilos en diferentes niveles que se van "sumando" o sobreescribiendo según su alcance
